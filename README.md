# كتاب الكيمياء - الصف الثالث الثانوي (تطبيق أوفلاين)

تطبيق أندرويد أوفلاين كامل لكتاب الكيمياء للصف الثالث الثانوي (193 صفحة).

## المميزات
- ViewPager2 للتمرير السلس
- Glide لعرض الصور بكفاءة
- أوفلاين 100%
- بناء آلي عبر GitHub Actions

## كيفية الاستخدام

1. أضف الصور في المجلد:
   `app/src/main/assets/images/`

2. ارفع المشروع على GitHub

3. سيتم بناء الـ APK تلقائياً
